📋 步骤1：创建飞书应用

🔗 打开飞书开放平台

访问 open.feishu.cn/app，使用飞书账号登录

登录后，点击右上方「开发者后台」

📝 创建企业自建应用

点击「创建企业自建应用」，填写：

  • 应用名称
  • 应用描述
  • 应用图标（可选）

🔑 获取凭证

在「凭证与基础信息」页面复制：

App ID (cli_xxx)
App Secret
⚠️ 重要：妥善保管 App Secret，不要分享给他人！

🔓 开启权限

进入「权限管理」，点击「批量导入」，粘贴以下 JSON：

{"scopes":{"tenant":["im:message","im:message:send_as_bot","im:message:readonly","im:message.p2p_msg:readonly","im:message.group_at_msg:readonly","im:message.group_msg","im:chat","im:chat.members:bot_access","im:resource","contact:user.employee_id:readonly"],"user":["im:chat.access_event.bot_p2p_chat:read"]}}

🤖 启用机器人能力

进入「应用能力」→「机器人」，开启机器人能力

勾选后才能进入下一步

⚙️ 步骤2：配置 OpenClaw 飞书凭证 & 绑定智能体

🔑 填入你的飞书凭证

建议和智能体同名，此 ID 将用于绑定智能体

🔍 不知道自己有哪些智能体？

点击下方按钮，复制提示词发给 OpenClaw，获取你的智能体列表：

点击解析按钮，从列表中选择一个

🤖 填入智能体信息

📋 生成的配置提示词

包含凭证配置 + 智能体绑定，点击复制发给 OpenClaw 执行：

# 任务：帮我配置飞书渠道 + 绑定智能体

## 我已有的信息
- 飞书 App ID: cli_xxx
- 飞书 App Secret: xxx
- 飞书机器人账号 ID: feishu-robot
- 智能体 ID: my-agent
- 智能体名称: 我的AI助手
- Workspace: /Users/用户名/.openclaw/workspace-myagent

## 请执行以下命令

### 第一步：配置飞书渠道
在终端运行以下命令配置飞书渠道：
```
openclaw channels add
```

选择 Feishu，然后输入：
- App ID: cli_xxx
- App Secret: xxx

### 第二步：绑定智能体
在 openclaw.json 中添加以下配置：

#### 1. 在 agents.list 中添加智能体（如不存在）：
```json
{
  "id": "my-agent",
  "name": "我的AI助手",
  "workspace": "/Users/用户名/.openclaw/workspace-myagent"
}
```

#### 2. 在 bindings 中添加绑定：
```json
{
  "bindings": [
    {
      "agentId": "my-agent",
      "match": {
        "channel": "feishu",
        "accountId": "feishu-robot"
      }
    }
  ]
}
```

### 第三步：重启网关
```
openclaw gateway restart
```
⚠️ 关键：必须先完成此步骤（凭证配置 + 智能体绑定 + restart），才能进入步骤3配置事件与回调！

勾选后才能进入下一步

📡 步骤3：配置飞书事件与回调 & 发布机器人

⚠️ 前置条件：必须先完成步骤2（配置凭证、绑定智能体并重启网关），才能配置事件与回调！

🌐 进入事件与回调页面

在飞书开放平台，进入你的应用 → 「事件与回调」

🔗 选择长连接模式

选择「使用长连接接收事件」（WebSocket 模式）

💡 不需要公网 URL，OpenClaw 自动处理连接

➕ 添加事件

点击「添加事件」，搜索并添加：

im.message.receive_v1 （接收消息 - 必须）
im.message.message_read_v1 （消息已读 - 可选）

🚀 发布机器人

完成事件配置后，还需要发布机器人才能使用：

1

点击「创建版本」

2

填写应用版本号（如 1.0.0）和更新说明

版本号格式：主版本.次版本.修订号（如 1.0.0）

勾选后才能进入下一步
