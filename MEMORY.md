## instructions-web 技能设计规范

### 交互功能要求
- 动态输入 Agent 名称，实时更新到页面各处
- 手动同步按钮（点击后才更新，不是自动）
- 一键复制按钮（避免鼠标拖拽）
- JSON 配置区独立复制按钮
- Toast 弹出反馈（已复制/已同步）
- 支持添加/删除智能体（2个或更多）

### 内容调整要求
- 创建智能体时自动创建 workspace，无需手动
- 配置参数说明表格（配置项/推荐值/作用）
- 添加 maxPingPongTurns 参数（控制轮数）
- 第二步标注"每个 agent 加 messaging"

### 视觉风格要求
- 苹果风格设计
- 浅绿色 + 紫色配色
- 粘性导航栏 + 快速跳转
