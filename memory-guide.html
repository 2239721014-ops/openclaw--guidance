开始配置

只需 3 步：

1 打开终端 → 创建记忆文件
cd ~/.openclaw/workspace-[你的agent名] && touch MEMORY.md && mkdir -p memory
2 复制 MEMORY.md 模板（见下方）
写入你的基本信息、偏好等
3 重启 Gateway
openclaw gateway restart

一键配置提示词

# 任务：帮我配置 OpenClaw 长期记忆

## 我需要什么
让我的 AI 助手能记住：
- 我是谁、怎么称呼我
- 我喜欢什么、不喜欢什么
- 之前我们聊过的重要事情

## 帮我做以下事情

### 1. 创建记忆文件
在 ~/.openclaw/workspace-[我的agent名]/ 目录下：
- 创建 MEMORY.md 文件
- 创建 memory/ 文件夹

### 2. 写入模板内容
帮我生成一份 MEMORY.md 模板，包含：
- 我的名字、称呼方式
- 我的偏好（沟通风格、信息密度等）
- 我的背景（可选）
- AI 助手的角色设定

### 3. 配置加载规则
根据 AGENTS.md 的规范，确保：
- 主会话（一对一）：加载 MEMORY.md + memory/
- 群聊：不加载 MEMORY.md（保护隐私）

### 4. 重启生效
执行 openclaw gateway restart

完成后告诉我！"

📋 配置参数说明

配置项 作用
MEMORY.md 长期记忆文件，存重要信息
memory/ 日常记录文件夹
加载规则 主会话加载，群聊不加载

常见问题

Q: 群聊也能用记忆吗？
→ 可以用，但只加载 memory/，别加载 MEMORY.md！否则你的隐私会被群友看到。
Q: 记忆文件能存多少？
→ 建议 MEMORY.md 控制在 2000-5000 字，太长 AI 会"记不住重点"。
Q: AI 怎么知道要"记住这个"？
→ 直接告诉它："把这个记下来"，它会自动写入 memory/ 或更新 MEMORY.md。
Q: 能同时运行多个有记忆的 AI 吗？
→ 可以！每个 agent 有独立的 workspace，互不影响。
